Margins & Paddings

Predefined classes to change the indentation.

Margins
margin-0
Remove all margins from element
margin-top-*
Use it to change top margin. Available *: 0, 5, 15, 20, 30, 40, 50
margin-bottom-*
Use it to change bottom margin. Available *: 0, 5, 15, 20, 30, 40, 50
margin-left-*
Use it to change left margin. Available *: 0, 5, 15, 20, 30, 40, 50
margin-right-*
Use it to change right margin. Available *: 0, 5, 15, 20, 30, 40, 50
Paddings
padding-0
Remove all paddings from element
padding-*
Use it to change padding. Available *: 0, 5, 15, 20, 30, 40, 50
padding-(*1)-(*2)
Use this structure to change element paddings.
(*1): left, right, top, bottom; (*2): 0, 5, 10, 15, 20, 30, 40, 50;
